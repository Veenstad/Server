# esx_CivAutoRepair
TAR Version 1.5 Release

### Requirements
* ESX
  * [esx](https://github.com/ESX-Org)

## Installation
- Add this in your server.cfg :

```
start esx_CivAutoRepair
```
-  * This release if for people who have the mechanic job you can remove this at any point you wish.
